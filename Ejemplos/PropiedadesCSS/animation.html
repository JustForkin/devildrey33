En este ejemplo se utiliza el mismo keyframe para las dos animaciones, pero se comportan de forma distinta porque se ha definido la dirección normal en uno, y alternate(invertir) en el otro.

NOTA : Este ejemplo no funciona en Internet Explorer 9 e inferiores!

Este objeto aparece y desaparece gradualmente. "animation:MiAnimacion 5s ease 0 infinite alternate"

Este objeto aparece gradualmente, y desaparece de golpe. "animation:MiAnimacion 5s ease 0 infinite normal"
