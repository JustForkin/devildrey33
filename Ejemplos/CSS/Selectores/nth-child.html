Ejemplo utilizando una fórmula code:nth-child(an+b) , 'a' es el avance, y 'b' es el inicio (por defecto es 0 y es un valor opcional).

Siempre se empieza a contar desde 0, por lo que si hacemos (4n), pasará directamente al objeto 4, 8, 12, etc...
Si queremos que empieze desde el primero '1, 5, 9, 13, etc..' debemos especificar el valor 'b' (4n+1).
