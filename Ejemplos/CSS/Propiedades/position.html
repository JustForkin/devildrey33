static El elemento se posiciona ordenadamente según el flujo de datos, de arriba a abajo. No se puede mover el elemento.
absolute El elemento se posiciona relativamente a su primer padre que no sea estatico, y se puede colocar en cualquier parte del documento.
fixed El elemento se posiciona relativamente a la ventana del navegador.
(para este ejemplo la ventana del navegador representa unicamente la parte inferior, y no el código de la parte superior, ya que se está mostrando con un iframe)
relative El elemento se posiciona relativamente a su posicion original, con la diferencia que podemos moverlo a partir de esa posición.
